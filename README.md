# 📊 Dashboard de Ventas – Tienda Retail  

**Autora:** Claudia Milena Balcarcel Palma
**Rol:** Data Analyst (Junior)  
**Herramientas:** Excel, Tablas Dinámicas, Gráficos, Diseño de KPIs  
**Dataset:** 1.000 transacciones  
**Año Analizado:** 2023  

---

## 🎯 Objetivo del Proyecto  
Analizar el rendimiento de ventas de una tienda retail mediante un dashboard interactivo en Excel, identificando patrones de compra por categoría, género y grupo etario.

---

## 📌 KPIs Incluidos  
- **Total Sales:** $456,000  
- **Total Transactions:** 1,000  
- **Average Order Value (AOV):** $456  

---

## 📈 Visualizaciones del Dashboard  
- **Sales by Category**  
- **Sales by Age Category**  
- **Sales by Gender**  
- **Average Value x Category**  

---

## 🔍 Principales Insights  
- El grupo **Middle Age** genera la mayor parte de las ventas.  
- La distribución por género es equilibrada (51% Female, 49% Male).  
- La categoría **Beauty** tiene el ticket promedio más alto.  
- Electronics y Clothing lideran en ventas totales.
